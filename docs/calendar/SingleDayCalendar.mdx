# Calendar - Single day

## File: SingleDayCalendar.tsx

This is a calendar that only spans one day, which day it is can be controlled by the 'date' prop. It is essentially a single TimeSlotColumn and the Gutter.

## Props

<Props of={SingleDayCalendar} />

## Usage

<Playground>
  {() => {
    const date = new Date();
    const min = new Date(date.setHours(7, 0, 0, 0));
    const max = new Date(date.setHours(20, 0, 0, 0));
    return (
        <SingleDayCalendar date={new Date()} min={min} max={max} columns={['test1', 'test2']} />
    )
  }}
</Playground>
