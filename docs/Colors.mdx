# Colors

These are the colors from the oslo color scheme that the designers use.

NB: Do not use the hex codes directly in the code, use the enum 'Colors' (found in the types file) instead, so f.ex. Colors.Red.

## Hex codes

<Playground>
  {() => {
    Colors = {
      DarkBlue: '#2A2859',
      Blue: '#6FE9FF',
      LightBlue: '#B3F5FF',
      DarkGreen: '#034B45',
      Green: '#43F8B6',
      LightGreen: '#C7F6C9',
      Red: '#FF8274',
      Yellow: '#F9C66B',
      LightBeige: '#F8F0DD',
      DarkBegie: '#D0BFAE',
      Black: '#2C2C2C',
      White: '#FFFFFF',
    }
    return null;
  }}
</Playground>
