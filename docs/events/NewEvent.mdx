# Event - New event

Component used in a Modal to create a new event. It's build from the vertical template and options.

## Props

<Props of={NewEvent} />

## Usage

<Playground>
  {() => {
    const modal = useModal();
    const showEvent = () => {
        modal.show(
            <NewEvent
                start={new Date()}
                end={new Date()}
            />
        )
    }
    return (
        <button onClick={() => showEvent()}>Show new event ui</button>
    )
  }}
</Playground>
