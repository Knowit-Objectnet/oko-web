# Event - Extra event

## File: ExtraEvent.tsx

Component used in a Modal to create an extra event / pickup. It's build from the vertical template and options.

## Props

<Props of={ExtraEvent} />

## Usage

<Playground>
  {() => {
    const modal = useModal();
    const showEvent = () => {
        modal.show(
            <ExtraEvent
                start={new Date()}
                end={new Date()}
            />
        )
    }
    return (
        <button onClick={() => showEvent()}>Show new pickup ui</button>
    )
  }}
</Playground>
